- sections: Array containing section objects.

- text: Object containing the question text mapped by language keys (en and hi).

- rubric: Always return an empty array.

- options: Optional array of objects used ONLY for "MCQ" types. Each object must have an `optionId` (e.g., "A", "B", "i") and a bilingual `text` object. - globalChoices: Array of objects for choices specifically between sections. Leave empty if no inter-section choices exist.
